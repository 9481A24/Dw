🌐 Curso: O que é IP e Porta

📍 O que é IP?

O IP (Internet Protocol) é como o endereço do seu aparelho na rede. Pode ser interno (dentro do Wi-Fi) ou externo (visto por sites).

  • IP Interno: Só dentro da sua casa. Ex: 192.168.0.105
  • IP Externo: Mostra seu endereço na internet. Ex: 177.20.50.88

🚪 O que são Portas?

Portas são como "portinhas" no seu aparelho por onde passam os dados.

  • Porta 80: Sites comuns (HTTP)
  • Porta 443: Sites seguros (HTTPS)
  • Porta 22: Acesso remoto (SSH)

💡 Curiosidade

Cada vez que você entra num site, seu aparelho usa um IP e uma porta para se comunicar com ele!

🛠️ Descobrindo seu IP no Termux

# IP Interno:
ifconfig

# IP Externo:
curl ifconfig.me
